# Images Directory

This directory should contain the following high-quality, royalty-free images:

- **fishfarm-1.jpg** - Aerial view of fish farm
- **fishfarm-2.jpg** - Close-up of fish pens
- **pump-industrial-1.jpg** - Industrial pump
- **pump-system-1.jpg** - Pipes/pump installation

## Image Requirements

- High resolution (minimum 1920px width recommended)
- Royalty-free / properly licensed
- Related to aquaculture and industrial pumps
- Optimized for web (compressed but high quality)

## Current Status

The website currently uses Unsplash URLs as placeholders. Replace these with local files by updating the image paths in `app/page.tsx` from:
- `https://images.unsplash.com/...` 
to:
- `/images/fishfarm-1.jpg` (or appropriate filename)
